# CosmoSearch - Specialized Space Search Engine

A Streamlit-based space search engine utilizing an Inverted Index and TF-IDF ranking system.

## Features
- Custom Inverted Index construction.
- Boolean retrieval supporting AND/OR query processing.
- Relevance ranking using TF-IDF.
- Interactive Web GUI with keyword highlighting.

## Setup & Installation Instructions
1. Install the required libraries:
```bash
   pip install streamlit wikipedia

2. Run the application:
```bash
   streamlit run Searchengine.py
